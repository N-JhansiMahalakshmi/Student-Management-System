About Student Management System

The Student Management System is a web-based application developed using Django, HTML, CSS, Bootstrap and SQL.

This system helps institutions efficiently manage student records including student details, departments, courses, reports and academic information.

Features included in this project:

  • Add Student Records
  • View Student Details
  • Delete Student Records
  • Upload Student Photos
  • Responsive Dashboard Design
  • Django Admin Panel
  • Bootstrap User Interface
